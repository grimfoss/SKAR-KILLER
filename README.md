# SKAR KILLER

This repository contains a list of Instagram usernames that use audio by SKARS.

The usernames were collected from `reels/audio` pages and extracted from Apify scrape results.

This is not a definitive list, but it should capture many of the accounts using SKARS audio.

Future contributions to this list are appreciated, whether by expanding the set of SKARS audio pages, adding other AI artist `reels/audio` pages, or growing the list of usernames.
